# Проект Repomix

Этот проект содержит конфигурационные файлы и файлы сборки для инструмента Repomix. Проект не содержит исходных файлов.

## Файлы

- **.gitignore**: Файл для указания игнорируемых файлов и директорий в Git. В данном случае игнорируются директории `node_modules/`, `out/`, а также файлы `.vsix`, `.DS_Store`, `.log`, `.ilnsk`, `.env` и `.gradle`.

- **build.gradle.kts**: Файл конфигурации сборки проекта на языке Kotlin. В нем указаны плагины для сборки Java-приложения.

- **repomix.config.json**: Конфигурационный файл для инструмента Repomix. В нем указаны параметры ввода и вывода, а также настройки безопасности и подсчета токенов.

## Использование

Для использования проекта необходимо установить Node.js и Kotlin. Далее можно запустить команду сборки проекта с помощью Gradle.
