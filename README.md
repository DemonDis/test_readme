# Проект Repomix

Этот проект содержит конфигурационный файл `repomix.config.json`, который настраивает работу инструмента Repomix. В конфигурации указаны параметры вывода, включаемые и исключаемые файлы, а также параметры безопасности.

Также в проекте есть файл `.gitignore`, который определяет файлы и директории, которые следует игнорировать при Commit в Git.

Единственный файл кода в проекте - `main.go`, содержащий простую программу на Go, которая выводит приветствие ("Hello, Go").

Проект не содержит других исходных файлов или директорий.

## Структура проекта

```
repomix-project/
│
├── .gitignore
│
├── main.go
│
└── repomix.config.json
```
