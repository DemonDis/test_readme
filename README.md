# Проект Repomix

Этот проект содержит конфигурационный файл `repomix.config.json`, который определяет настройки для инструмента Repomix. В этом файле указаны параметры ввода и вывода, а также настройки безопасности и подсчета токенов.

Также в проекте присутствует файл `.gitignore`, который определяет файлы и директории, которые следует игнорировать при коммите в Git.

Единственный файл кода в проекте - `main.go`, который содержит простую программу на Go, выводящую приветствие и информацию о пользователе.

## Установка и запуск

1. Клонируйте репозиторий:
```
git clone https://github.com/yourusername/repomix.git
```
2. Перейдите в директорию проекта:
```
cd repomix
```
3. Запустите программу:
```
go run main.go
```
